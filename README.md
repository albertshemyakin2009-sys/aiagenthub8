# AIAGENTHUB — Static Demo (Theme + Requests + Auth)

Обновлённый прототип маркетплейса AI-агентов.

## Что внутри

- **Главная (`index.html`)**
  - Hero-блок, live-панель агентов, быстрые CTA.
  - Избранные агенты с раскрывающимися описаниями.

- **Каталог (`catalog.html`)**
  - Поиск по названию/нише.
  - Фильтры по категориям.
  - Карточки агентов:
    - краткое описание,
    - кнопка «Подключить» (просит авторизацию),
    - кнопка «Подробнее» — раскрывает use case, пример запроса, интеграции и условия.

- **О нас (`about.html`)**
  - Миссия, для кого платформа, качество, планы.
  - Внизу — мокап: планшет на диване с открытой главной страницей,
    визуально согласован с темой (dark / light), реализован в чистом CSS.

- **Техническая поддержка (`support.html`)**
  - Отдельный канал: баги, доступ, проблемы с агентами.
  - Форма (демо) с алертом.

- **Запросы и пожелания (`requests.html`)**
  - Предложить нового агента.
  - Партнёрство, улучшения платформы, идеи.
  - Отдельно от техподдержки.

- **Настройки (`settings.html`)**
  - Блок «Профиль» (имя / команда, email) как точка входа в персонализацию.
  - Переключатель языка (RU / EN).
  - Переключатель темы:
    - Оригинальная (тёмная),
    - Светлая (мягкий серо-белый UI).
  - Выбор валюты.
  - Все действия в демо-режиме (локальное состояние).

- **Админ-панель (`admin.html`)**
  - Прототип модерации: сводка, список агентов, статусы live/review/draft.

- **`script.js`**
  - мок-данные агентов,
  - логика каталога и фильтров,
  - раскрытие «Подробнее»,
  - разделённые формы Support / Requests,
  - i18n RU/EN,
  - модальное окно регистрации/входа + «Продолжить как гость»,
  - переключение темы (с сохранением в localStorage),
  - демо-подключение агентов.

- **`styles.css`**
  - Единая дизайн-система на CSS-переменных.
  - Поддержка двух тем:
    - оригинальная (глубокий тёмный техно-стиль),
    - светлая (чистый нейтральный интерфейс).
  - Адаптив под мобильные (исправлены наложения блоков).
  - CSS-мокап планшета на диване в духе бренда.

## Запуск

1. Открыть `index.html` напрямую
   или
2. В корне:

```bash
python3 -m http.server 4173
# или
npx serve .
```
